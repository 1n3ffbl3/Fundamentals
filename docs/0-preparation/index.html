0. Preparation

Preparation

Before moving on to the first project, there are a few pre-requisites:

  • JavaScript fundamentals
  • HTML/CSS fundamentals
  • A portfolio hosted on Gh-Pages
  • 8 completed JS 30 challenges, each with:
    • A GitHub repo
    • A good, descriptive readme
    • A live Gh-Pages demo

Everything you need to get started can be found in the Elewa Precourse. So follow this link and complete all 3 projects in our precourse.

You can find an example of the portfolio you'll be building right here.

Studying tips

Before diving right in (and every other day of course!), you should check out this video about learning to program. Also take a look at Thomas Frank's youtube channel. It's full of great tips about we learn and how to study effectively.
